## デザイン仕様

- **背景色**: `#FFFFFF`（白）と `#17bbef`（ブランドカラー）を交互に使用
- **アクセントカラー**: `#17bbef`
- **文字色**: `#333333`
- **フォント**: Noto Sans JP（Google Fonts）
- **アイコン**: Font Awesome 6
- **写真**: グレープレースホルダー（`[写真]`）で代替

## アニメーション

- ヒーローセクション：Canvas を使った浮遊パーティクル（白・透明度変化）
- スクロールフェードイン：IntersectionObserver による `.fade-in / .fade-in-left / .fade-in-right`
- 導入効果：数値カウントアップアニメーション
- カードホバー：`transform: translateY(-4px)` + `box-shadow` 拡大
- FAQアコーディオン：`max-height` トランジション
- バッジ：`pulse` キーフレームアニメーション
- プレースホルダー：シマーアニメーション
